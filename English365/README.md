# English365

> Học tiếng Anh mỗi ngày — 365 ngày, 365 bài học.

English365 là module học tiếng Anh trong hệ thống Project365. Mỗi ngày bạn sẽ có một **Lesson Pack** gồm các tài liệu học tập được thiết kế có cấu trúc.

## Cấu trúc Lesson Pack

Mỗi Day (001, 002, ..., 365) gồm 3 file bắt buộc:

| File | Mô tả |
|------|-------|
| `README.md` | Tổng quan: topic, vocabulary count, grammar, status |
| `Lesson.md` | Nội dung bài học chính (Goal, Vocabulary, Grammar, Reading, Listening, Speaking, Writing, Pronunciation, Shadowing, Review, Reflection) |
| `Anki.csv` | Flashcard để import vào Anki |

## Quy trình học mỗi ngày

**Morning:**

1. 🎯 **Goal** — Đọc mục tiêu bài học
2. 📚 **Vocabulary** — Học từ vựng mới
3. 📝 **Grammar** — Nắm cấu trúc ngữ pháp
4. 📖 **Reading** — Đọc đoạn văn mẫu
5. 🎧 **Listening** — Nghe và chép lại
6. 🗣 **Speaking** — Luyện nói
7. ✍ **Writing** — Luyện viết

**Evening:**

8. 🎤 **Pronunciation** — Luyện phát âm
9. 🧠 **Shadowing** — Nghe và nhắc lại
10. 📅 **Review** — Ôn tập kiến thức
11. 📓 **Reflection** — Tự đánh giá
12. **Anki** — Ôn flashcard
