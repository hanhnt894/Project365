# PROJECT365 Guide

> User manual for the Project365 self-development system.

## Morning Workflow

1. Open terminal in the repository root.
2. Run `git pull` to sync with remote.
3. Open yesterday's lesson for a quick review (5 min).
4. Check today's day number from Progress.md.
5. Set a specific goal for today's session.

## Learning Workflow

1. **Goal** — Read the lesson goal, understand what you will achieve.
2. **Vocabulary** — Learn maximum 10 new words. Write each word, its meaning, and an example sentence.
3. **Grammar** — Read the grammar point, write your own example.
4. **Reading** — Read the passage aloud, note unfamiliar phrases.
5. **Listening** — Listen to audio (if available), write what you hear.
6. **Speaking** — Practice saying key sentences aloud. Record yourself if possible.
7. **Writing** — Complete the writing exercise without assistance first, then check.

## Night Workflow

1. **Pronunciation** — Practice difficult sounds. Use IPA if needed.
2. **Shadowing** — Listen and repeat simultaneously, focusing on intonation.
3. **Review** — Go through all new vocabulary and grammar from today.
4. **Reflection** — Answer Reflection questions. Be honest about difficulties.
5. **Anki** — Import `Anki.csv` and review flashcards.
6. **Log errors** — Record every mistake in ErrorLog (inside Lesson.md or a dedicated file).

## Weekly Review

Every Sunday:

1. Open the WeeklyReview directory.
2. Create a review document: `WeeklyReview/Week-XX.md`.
3. List all vocabulary learned this week.
4. Identify recurring mistakes from ErrorLog.
5. Rate your consistency (1-5).
6. Plan focus areas for next week.

## Monthly Review

Last calendar day of the month:

1. Create a review document: `MonthlyReview/YYYY-MM.md`.
2. Review all weekly reviews from the month.
3. Update overall Progress.md.
4. Adjust learning strategy if needed.
5. Set monthly goals for the next month.

## How to Work with AI Coach

- The AI Coach generates Lesson Packs based on a daily topic.
- You provide the topic or let the AI choose progressively.
- The AI Coach creates: `Lesson.md`, `Anki.csv`, and `Reflection` questions.
- Review AI-generated content critically — you are the ultimate decision-maker.
- Ask the AI Coach for:
  - Grammar explanations
  - Example sentences
  - Pronunciation guidance
  - Error correction
  - Review exercises
- Treat the AI Coach as a tutor, not a crutch — always try first before asking.
