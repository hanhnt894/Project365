# Review Rules

## Weekly Review

- **When:** Every Sunday
- **What:**
  - Review all vocabulary learned during the week
  - Analyze ErrorLog for recurring mistakes
  - Rate consistency and progress (1-5 scale)
  - Plan focus areas for the next week
- **Output:** `WeeklyReview/Week-XX.md`

## Monthly Review

- **When:** Last calendar day of the month
- **What:**
  - Aggregate all weekly reviews
  - Update Progress.md
  - Assess if learning goals are being met
  - Adjust strategy if necessary
  - Set next month's goals
- **Output:** `MonthlyReview/YYYY-MM.md`

## Sprint Review

- **When:** Every 4 learning days
- **What:**
  - Review sprint goal vs actual progress
  - Identify blockers
  - Retrospective: what went well, what can be improved
  - Plan next sprint
- **Output:** Notes in documentation or directly in the repository.
